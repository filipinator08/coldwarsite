# Cold War Influence Website

This site compares the influence strategies of the U.S. and U.S.S.R. during the Cold War.

## How to Use
- Open `index.html` in any browser.
- Host on GitHub Pages or Netlify to publish online.
- Keep `assets/` folder structure intact.

## Features
- Proxy war coverage: Korea, Angola, Nicaragua
- Primary sources, videos, podcasts
- Interactive timeline, quiz, and animation

Created for educational purposes.
